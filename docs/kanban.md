# Tablero Kanban del Proyecto

El proyecto utiliza un tablero Kanban en GitHub Projects para organizar las actividades del desarrollo.

## Columnas utilizadas

- Backlog
- To Do
- In Progress
- Review
- Done

## Propósito

El tablero permite dar seguimiento a las tareas del proyecto, asignar responsables, controlar el avance de los issues y evidenciar el trabajo colaborativo del equipo.

## Evidencia

Se agregan capturas del tablero Kanban dentro de la carpeta evidencias.
